# Azure VM Setup

This document provides a high-level overview of the Azure virtual machine used in the lab.

## Platform

The lab uses an Ubuntu Server virtual machine deployed in Microsoft Azure.

## Purpose

The VM is used for:
- Linux administration practice
- cloud infrastructure learning
- k3s deployment
- Argo CD testing
- ingress and service publishing experiments

## Main Areas

- Azure resource deployment
- virtual machine administration
- networking basics
- remote access
- system updates and maintenance

## Notes

This repository intentionally excludes sensitive configuration details such as public IP addresses, credentials, private keys and tenant-specific information.
