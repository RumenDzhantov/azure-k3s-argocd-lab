# k3s Installation

This document provides a high-level overview of the k3s deployment used in the Azure lab.

## Purpose

k3s is used in this lab to provide a lightweight Kubernetes environment for learning and experimentation.

## Main Use Cases

The k3s environment is used for:
- Kubernetes basics
- workload deployment
- service exposure
- ingress testing
- GitOps-related experiments with Argo CD

## Learning Goals

The k3s setup helps build practical skills in:
- cluster installation
- basic Kubernetes resource management
- service and ingress concepts
- troubleshooting deployment issues

## Notes

This repository focuses on documentation and architecture at a high level. Sensitive operational details are not included.
