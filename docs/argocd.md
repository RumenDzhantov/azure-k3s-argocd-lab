# Argo CD

This document provides a high-level overview of how Argo CD is used in the Azure k3s lab.

## Purpose

Argo CD is used in the lab to explore GitOps-oriented deployment and Kubernetes application management.

## Main Areas

Argo CD is used for:
- application deployment management
- Kubernetes resource synchronization
- Git-based configuration workflows
- learning GitOps concepts in practice

## Learning Value

Working with Argo CD helps build practical knowledge of:
- declarative infrastructure ideas
- Kubernetes application delivery
- deployment visibility
- troubleshooting sync and configuration issues

## Notes

This repository documents the lab at a high level and does not include sensitive configuration details, secrets or private repository information.
