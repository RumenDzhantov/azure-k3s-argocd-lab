# Azure k3s Argo CD Lab

This repository documents my Azure-based lab environment built on Ubuntu Server, k3s, Traefik and Argo CD.

## Project Goals

- document a practical cloud lab in Microsoft Azure
- show Linux server deployment and administration
- demonstrate Kubernetes basics with k3s
- document ingress and GitOps-related components
- build a portfolio project for internships and IT infrastructure roles

## Core Technologies

- Microsoft Azure
- Ubuntu Server
- k3s
- Kubernetes
- Traefik
- Argo CD
- Cloudflare DNS / service publishing
- Linux administration

## Environment Overview

The lab is based on an Ubuntu Server virtual machine deployed in Azure. It is used for Linux administration, Kubernetes practice and cloud infrastructure experimentation.

Main focus areas:
- Azure VM deployment
- Ubuntu Server administration
- k3s installation and management
- Traefik ingress
- Argo CD setup
- service exposure and troubleshooting
